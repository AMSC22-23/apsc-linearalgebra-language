Detailed Description

template<class Matrix, class Vector, ORDERINGTYPE ORDER_TYPE>
class apsc::LinearAlgebra::MPIFullMatrix< Matrix, Vector, ORDER_TYPE >

A class for parallel matrix product.

Template Parameters
MatrixA matrix compliant with that in Matrix.hpp
VectorThe vector type used for the local solution and internal usages. It must have the following methods:
  • data(): Returns a pointer to the data buffer
  • resize(std::size_t): Resizes the vector with the requested length It must have also a range constructor as std::vector

All processes will call setup but only the managing process (with number 0) will actually pass a non-empty global matrix

Parameters
gMatThe global matrix
communicThe MPI communicator
Note
This is not the only way to create a parallel matrix. Alternatively, all processes may directly build the local matrix, for instance reading from a file. In this case, for the setup we need just the number of rows and columns of the global matrix
