Detailed Description

template<typename EigenStructure, typename Scalar, typename MappedMatrix = Scalar *>
class apsc::LinearAlgebra::EigenStructureMap< EigenStructure, Scalar, MappedMatrix >

A full Eigen-compatible matrix class with custom-handled buffer data. This class is meant to call Eigen methods on data not owned by Eigen, in order to avoid memory movements.

Template Parameters
EigenStructureThe mapped Eigen type (e.g., MatrixX<>, VectorX<>)
ScalarThe scalar type
MappedMatrixThe custom matrix type that owns the data buffer
