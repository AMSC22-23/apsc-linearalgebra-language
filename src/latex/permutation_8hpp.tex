Create permutation matrices based on row and column indices. 

This function creates permutation matrices based on row and column indices. It generates row permutation matrix {\ttfamily Pr} and column permutation matrix {\ttfamily Pc} using the given row and column indices {\ttfamily I} and {\ttfamily J}, respectively. The size of the matrices {\ttfamily Pr} and {\ttfamily Pc} is determined by the dimensions {\ttfamily n1} and {\ttfamily n2}.
