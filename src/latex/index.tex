\hypertarget{index_autotoc_md4}{}\doxysection{Compilation}\label{index_autotoc_md4}
This repository is intended to be used inside the {\ttfamily pcafrica/mk} docker image env. See \href{https://github.com/HPC-Courses/AMSC-Labs/tree/main/Labs/2023-24/lab00-setup}{\texttt{ here}} for the configuration. If so, you don\textquotesingle{}t need to take additional steps.

In case you want to run the project locally, you have to\+:
\begin{DoxyItemize}
\item Export the {\ttfamily Eigen} include directory path under the env variable {\ttfamily \$\+E\+I\+G\+E\+N3\+\_\+\+I\+N\+C\+L\+U\+D\+E\+\_\+\+D\+IR}.
\item Have {\ttfamily M\+PI} installed (Note\+: we had no issued using version {\ttfamily 3.\+1} of {\ttfamily M\+PI}, newer versions might not compatible)
\end{DoxyItemize}

\begin{DoxyCode}{0}
